# BattleSimulation

## 目录结构
common 算法所需的共用文件  
model 训练模型文件  
data  生成环境所需要的数据源  
environment  交互环境目录  
---arguments.py 用于配置环境相关信息的参数文件  
---battle_simulation.py 定义环境交互函数文件  
---enemy.py 敌军目标类  
---utils.py 常用函数文件
